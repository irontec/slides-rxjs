## Multicast / Unicast

Las promesas se resuelven sin tener en cuenta cuantos callbacks estÃ¡n conectados (o desde cuando):

```javascript
// T0
const p = new Promise((resolve) => {
  setTimeout(()=>resolve(1), 1000);
});

p.then(value=>console.log("uno", value)); // uno 1 - T0 + 1000ms

setTimeout(() => {
  // se genera el callback en T0+500ms
  p.then(value=>console.log("dos", value)) // dos 1 - T0 + 1000ms
}, 500);
```
