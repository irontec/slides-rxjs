## Subscription

```typescript
const obs = fromEvent(window,'click');
const sub = obs.subscribe((e:Event) => {
  console.log(`La coordenada X es: ${e.clientX}`);
});

setTimeout(() => {
    sub.unsubscribe();
}, 3000);
```
